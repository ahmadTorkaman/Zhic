# Zhic — 3D asset request (bedroom sets)

**Date:** 2026-06-08 · **For:** 3D artist · **From:** Zhic web/catalog

We need two batches of renders for the website. **Total: 41 room scenes + 6 product photos.**

---

## File specs (please follow exactly — filenames are used directly by the site)

**Batch A — per-age room SET SCENES**
- Format: **WebP**, portrait **1200 × 1707 px** (3:4-ish), like the existing
  `bedroom-set-parla-baby-card.webp`.
- Content: a styled scene/collage of **one bedroom set, configured for ONE age group**.
- Filename: **`bedroom-set-<series>-<age>-card.webp`** (ages: `baby`, `teen`, `double`, `bunk`).

**Batch B — single PRODUCT photos**
- Format: **WebP**, **2700 × 2700 px**, **transparent background**, like the existing studio renders.
- Filename: keep the product's existing slug, e.g. `parla-bed-box.webp`.

### ⚠️ Most important rule for Batch A
Each age scene must show **only the furniture that belongs to that age's set.** Examples:
- **baby (نوزاد):** crib + changing unit + wardrobe etc. — **no study desk.**
- **teen (نوجوان):** single bed + study desk + bookcase etc.
- **double (دونفره):** double bed configuration.
- **bunk (دوطبقه):** the bunk bed set only.
We read which products belong to each age *from these scenes*, so the staging must be accurate.
