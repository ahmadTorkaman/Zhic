# Media completeness audit — catalog photography & hub media

**Date:** 2026-06-20  ·  **Scope:** all 26 designs (19 assetted + 7 image-less), 271 products, 36 categories, sections.

**Method:** Finder-tag inventory + box catalog cross-ref (deterministic gaps) **and** a 19-agent visual pass that opened and graded every product's best image (styled / clean / cutout / poor / missing). The visual pass is authoritative — Finder `picture` tags were applied inconsistently, so tags alone undercount/overcount quality.

**Quality bar:** the `/bedroom-furniture` mosaics want **styled** (in-room / lifestyle) imagery. `clean` = tidy studio render (acceptable fallback). `cutout` = bare object on white (works but flat — wants a styled shot). `poor` = defective, needs re-render.
