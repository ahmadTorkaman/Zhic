سرویس خواب نوزاد

نُه طرح برای نخستین اتاق — جایی برای رشد، نه برای بزرگ‌نمایی.

خانه / سرویس خواب / نوزاد

اولین اتاق چیزی است که نمی‌توان به آن بازگشت. ژیک نُه طرح برای این لحظه دارد — هر کدام با تخت قابل‌تبدیل، یا تخت نوزاد ساده، یا چارچوبی که از سن صفر تا هفت می‌ماند. هیچ‌کدام شلوغ نیستند. هیچ‌کدام ادعا ندارند که برای ابد. ولی هر کدام برای جایی که شب نوزاد در آن می‌خوابد، طراحی شده.

روکش‌ها از گردو و راش‌اند — هر دو با پوشش طبیعی، بدون رنگ شیمیایی. لبه‌ها مدور، گوشه‌ها بدون تیزی. هر طرح در همین صفحه قابل دیدن است؛ کلیک روی هر کارت شما را به صفحه‌ی کامل آن طرح می‌برد، جایی که قطعه‌های همراه — پاتختی، کمد، دراور — نشان داده می‌شود.
